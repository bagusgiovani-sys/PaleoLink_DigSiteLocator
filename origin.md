## Critical Findings (must-fix)

1. **[PERF]** `scientists`, `marketItems`, `digSites` arrays are declared inside `DigSiteLocator` — new array objects on every render. These are static data that never changes. — `src/App.tsx:17–320`
2. **[PERF]** `WeatherAnimation` is defined as an inner component inside `DigSiteLocator` — React loses component identity on every parent render, causing all 6 weather animation instances to unmount and remount. — `src/App.tsx:347–469`
3. **[PERF]** `calculateSafetyRisk`, `getWeatherColor`, `getWeatherIcon`, `getSiteIcon` are pure functions defined inside the component — re-created as new function objects every render. — `src/App.tsx:127–345`
4. **[PERF]** `resourceMap` (large data object) is declared inside the `.map()` callback — re-created for every matched site, every render. — `src/App.tsx:1285–1326`
5. **[PERF]** `Math.random()` in `WeatherAnimation` initial/animate/transition props — particle positions re-randomize on every component render. — `src/App.tsx:356–379`
6. **[ARCH]** `App.tsx` is 1,425 lines — all feature JSX (Scientists, Marketplace, Pathfinder, Expedition) inline. Section 2 (component split) is still pending. — `src/App.tsx`

3. **[DX]** `src/assets/react.svg` — Vite scaffold default, never imported anywhere.
